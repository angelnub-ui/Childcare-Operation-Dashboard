# Childcare-Operation-Dashboard
This project is an Excel-based operational dashboard created to analyze attendance, classroom performance, age group distribution, and behavioral incident trends in an early education setting.

## Key Insights
- Age group 2–3 recorded the highest number of incidents.
- Blue classroom showed the highest behavioral incident frequency.
- Absence rate was 25% of total students.
